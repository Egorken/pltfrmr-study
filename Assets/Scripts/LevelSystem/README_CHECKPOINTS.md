# Настройка чекпоинтов

Игрок при достижении чекпоинта запоминает точку; после смерти появляется там (без перезагрузки сцены).

---

## 1. Level Manager (один раз на сцену)

- В сцене должен быть объект с компонентом **Level Manager**.
- В поле **Player** перетащи объект игрока (или оставь пусто — при респавне игрок ищется по тегу **Player**).
- Level Manager можно сделать DontDestroyOnLoad, чтобы чекпоинт сохранялся при смене уровня (уже так в скрипте).

---

## 2. Чекпоинты на уровне

1. Создай пустой объект в нужном месте (там, откуда должен «запускаться» игрок).
2. Добавь **Collider 2D** (например Box Collider 2D) и включи **Is Trigger**.
3. Добавь компонент **Checkpoint**.
4. Опционально: создай дочерний объект с пустым Transform и перетащи его в **Spawn Point** — респавн будет в этой точке. Если **Spawn Point** пусто — используется центр объекта с Checkpoint.

Можно сделать несколько чекпоинтов: при прохождении каждого следующий перезаписывает точку респавна.

---

## 3. Игрок

- У объекта игрока должен быть тег **Player** (чтобы Level Manager и Checkpoint его находили).
- В **Player Health** включено **Respawn At Checkpoint** (по умолчанию включено) — тогда при смерти игрок переносится на последний чекпоинт и восстанавливает здоровье. Если выключить — при смерти сцена перезагрузится по-старому.

---

## Кратко

- **Level Manager** в сцене, у игрока тег **Player**.
- **Checkpoint** — объект с триггер-коллайдером (и при желании Spawn Point).
- После смерти игрок появляется на последнем пройденном чекпоинте с полным здоровьем.
